ГИСТОЛОГИЯ — ТРЕНАЖЁР (локальное приложение)
=============================================

КАК ЗАПУСТИТЬ
-------------
Просто открой файл  index.html  двойным кликом — он откроется в браузере.
Интернет не нужен, ничего устанавливать не надо.
(Рекомендуется Chrome, Edge, Firefox или Safari.)

ВХОД
----
Готовые учётные записи (логин / пароль):
    admin     / 1234
    student1  / histo1
    student2  / histo2
    student3  / histo3

Добавить/изменить пользователей: открой файл  users.js  любым текстовым
редактором и допиши строку  "логин": "пароль",  затем сохрани.

КАК РАБОТАЕТ ТЕСТ
-----------------
• Слева — микрофотография, справа — поле под каждую цифру выноски.
• Заполни поля и нажми «Проверить» (или Enter на последнем поле).
• Правильные подчёркиваются зелёным, ошибочные — красным.
• Под каждым полем показывается верный ответ + общий счёт.
• Ввод нестрогий: опечатки, сокращения и синонимы засчитываются.
• «Следующий (случайный)» — каждый раз новый случайный слайд.
• Переключатель «Все / Электронные / Препараты» и кнопка «перемешать».

Примечание про препараты (П-…): на самих фотографиях цифры не подписаны,
поэтому там нужно просто перечислить структуры по списку.

СОДЕРЖИМОЕ ПАПКИ
----------------
    index.html   — страница приложения
    styles.css   — оформление
    app.js       — логика теста
    data.js      — все слайды и ответы (сгенерировано из атласов)
    users.js     — учётные записи
    images/      — 108 изображений (P01…P62 препараты, E01…E46 электронные)

КАК ДОБАВИТЬ НОВЫЕ СЛАЙДЫ
-------------------------
Скинь мне новые файлы — я допишу их в data.js и добавлю картинки в images/.
Структура одной записи в data.js:
    {
      "id": "Э-47",
      "set": "Э",                 // "Э" или "П"
      "img": "E47.jpg",
      "title": "Название препарата",
      "sub": "увеличение / окраска",
      "answers": [ {"n":1,"text":"структура 1"}, {"n":2,"text":"структура 2"} ]
    }
Можно также добавить синонимы:  {"n":1,"text":"ядро","alt":["карион"]}
